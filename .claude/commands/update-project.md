# /update-project

Cập nhật PROJECT.md để phản ánh trạng thái phát triển hiện tại của dự án Smart POS.

## Các bước thực hiện

1. Đọc PROJECT.md hiện tại để nắm context
2. Đọc git log 20 commit gần nhất để biết những gì đã thay đổi
3. Scan các file source chính để xác nhận tính năng nào đã hoàn thành, đang làm, chưa làm
4. Cập nhật PROJECT.md:
   - Đánh dấu [x] cho tính năng đã hoàn thành
   - Thêm tính năng mới nếu có
   - Cập nhật Roadmap dựa trên hướng phát triển gần nhất
   - Cập nhật dòng "Cập nhật lần cuối" với ngày hôm nay
5. Commit PROJECT.md với message ngắn gọn

Không thay đổi nội dung nào khác ngoài PROJECT.md. Không tạo file mới.
