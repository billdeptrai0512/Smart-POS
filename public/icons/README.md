# App Icons

Place your app icons here with the following exact filenames and sizes:

| File               | Size      | Purpose                        |
|--------------------|-----------|--------------------------------|
| `icon-192x192.png` | 192×192px | Standard PWA icon              |
| `icon-512x512.png` | 512×512px | Splash screen & maskable icon  |

**Tips:**
- Use a square PNG with transparent background.
- For the maskable icon (512×512), keep the main logo within the center 80% "safe zone" so it doesn't get cropped on different devices.
- You can generate both sizes from a single high-res image using tools like [RealFaviconGenerator](https://realfavicongenerator.net/) or [PWA Asset Generator](https://github.com/nicedoc/pwa-asset-generator).
